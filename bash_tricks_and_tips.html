History

    Bash reads history off disk at start and writes complete history back on close.

  • Problem

    Bash history does not appear to be saved all the time. Multiple terminals create race conditions.

  • Solution

    Append to history & write history every time prompt is displayed.
    Add the following to .bashrc 

    shopt -s histappend
PROMPT_COMMAND='history -a'

History Contents

  • Increase storage size
    Add the following to .bashrc
    •   HISTSIZE=10000
  HISTFILESIZE=20000
  • Ignore duplicates and commands starting with whitespace.
    Add the following to .bashrc 
    HISTCONTROL=ignoredups:ignorespace

Searching History

  • Up/down arrows
  • Ctrl+R/S
  • Sane search with PgUp/PgDown.
    Add to .inputrc 
    "\e[5~": history-search-backward
"\e[6~": history-search-forward

Directories
